# 리워크케어(ReWorkCare) 디자인 개선 가이드

이 문서는 기존 MVP 디자인을 기반으로, 2025년 기준 신뢰 기반 스타트업 UI/UX 스타일로 개선하기 위한 정리본입니다. 색상(초록 기반)을 유지하면서도 현대적인 느낌을 살리는 방향을 중심으로 작성되었습니다.

---

## 1. 컬러 팔레트 개선

### 문제점

- 기존 초록 계열이 원색적이라 2010년대 웹 느낌을 줌
- 대비가 강해 디자인이 투박해 보임

### 개선 방향

- 초록 계열 유지 + 채도 낮추기 + 중립 배경 강화
- 의료·헬스케어 톤을 유지하면서도 Modern SaaS 스타일로 전환

### 추천 색상 팔레트

| 용도 | 색상 코드 | 설명 |
|------|----------|------|
| **Primary** | `#2F6E4F` | 톤다운 초록 (메인 액션, 강조 요소) |
| **Secondary** | `#61C48C` | 부드러운 서브 초록 (보조 액션, 상태 표시) |
| **Background Base** | `#F7F9F8` | 녹색 기운의 연그레이 (전체 배경) |
| **Container** | `#FFFFFF` | 흰색 (카드, 모달 배경) |
| **Border** | `#E4E7E7` / `#D0D5D4` | 연한 회색 (테두리, 구분선) |
| **Text Main** | `#1C1C1E` | 진한 회색 (본문 텍스트) |
| **Text Sub** | `#555555` | 중간 회색 (보조 텍스트) |

## 2. 타이포그래피 개선

### 문제점

- Paperozi가 UI 전반에 사용되면 전문성이 떨어짐
- 제목 대비가 부족하여 신뢰도가 떨어짐

### 개선 방향

- **Paperozi**: 로고 전용으로 제한
- **UI 전체**: Pretendard로 통일해 안정감 확보

### 텍스트 계층 구조

| 요소 | 크기 | 굵기 | 추가 스타일 |
|------|------|------|-------------|
| **H1** | 30px | 700 | letter-spacing: -0.5px |
| **H2** | 22px | 600 | - |
| **H3** | 18px | 600 | - |
| **Body** | 16px | 400 | - |
| **Caption** | 14px | 400 | - |

## 3. 레이아웃 개선

### 문제점

- 공간·간격·섹션 구분이 부족해 아마추어 느낌이 남
- 그림자 사용이 과하거나 일관되지 않음

### 개선 규칙

| 항목 | 규격 |
|------|------|
| **섹션 여백** | 상/하 48px |
| **카드 패딩** | 24px |
| **본문 최대 너비** | 680px (본문 중심 페이지) |
| **그림자** | 제거 → Border 중심 구조로 전환 |
| **아이콘 스타일** | 모두 Outline으로 통일 |

---

## 4. 버튼 스타일 개선

### 문제점

- 모든 버튼이 초록색이라 브랜드 톤이 과함

### 개선안

| 버튼 타입 | 색상 | 사용처 |
|-----------|------|--------|
| **Primary CTA** | `#2F6E4F` (초록) | 주요 액션 버튼 |
| **Secondary CTA** | Border `#D0D5D4` + 텍스트 진회색 | 보조 액션 버튼 |
| **Tertiary CTA** | 텍스트 버튼 `#4A4A4A` | 부가 액션 버튼 |

---

## 5. 카드 컴포넌트 개선

### 기존 문제

- 배경·텍스트·아이콘 규칙 부족 → 구식 느낌

### 개선된 카드 구조

```css
/* 카드 기본 스타일 */
.card {
  background: #FFFFFF;
  border: 1px solid #E4E7E7;
  border-radius: 12px;
  padding: 24px;
}

/* 아이콘 */
.icon {
  stroke-width: 1.75;
  color: #2F6E4F;
}

/* 제목 */
.title {
  font-size: 17px;
  font-weight: 600;
  color: #1C1C1E;
}

/* 본문 */
.body {
  font-size: 16px;
  color: #555555;
}

/* 간격 */
.spacing {
  /* 12px 단위로 통일 */
  gap: 12px;
  margin-bottom: 12px;
}
```

## 6. 병원 상세 보기 바텀시트 개선

### 문제점

- 정보가 크고 투박함

### 개선안

- **제목 + 태그칩 구조**: 정보를 시각적으로 그룹화
- **섹션 구분**: 각 섹션을 Border로 분리해 계층 명확화
- **버튼 배치**: Primary/Secondary 버튼을 나란히 2열로 정리
- **아이콘 스타일**: 모두 선형(Outline) 스타일 적용

---

## 7. 서류 안내 아코디언 개선

### 문제점

- 펼쳐지는 영역 정보량이 많은데 시각적 리듬 부족

### 개선안

- **카드 형태로 개선**: 아코디언을 카드 형태로 변경
- **열림 상태 배경**: `#F7F9F8`로 변경하여 시각적 구분
- **AI 요약 박스**: border만으로 구분 (그림자 제거)
- **텍스트 계층**: 내부 텍스트 계층 명확하게 정리

## 8. 전체 디자인 수준을 현대적으로 끌어올리는 핵심 요소

아래 5가지 규칙만 지켜도 현대적 SaaS 느낌을 바로 얻을 수 있습니다.

### 핵심 5가지 규칙

1. **초록 채도 줄이기**
   - 원색적 초록 → 톤다운 초록 (`#2F6E4F`)

2. **배경에 연그레이 계층 추가**
   - 순백색 → 녹색 기운의 연그레이 (`#F7F9F8`)

3. **그림자 최소화 → Border 체계로 전환**
   - 그림자 제거 → Border 중심 구조

4. **폰트 대비 강화 (Pretendard 중심)**
   - Paperozi는 로고 전용, UI는 Pretendard 통일

5. **아이콘 스타일 통일 (Outline)**
   - 모든 아이콘을 Outline 스타일로 통일
