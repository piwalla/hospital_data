## 2. ⚠️ 주요 주의사항 (Critical Alerts)

### 🚨 Supabase 프로젝트 일시정지 (Pausing)

- **현상**: Supabase Free Tier는 **1주일 동안 대시보드 접속이나 API 요청이 없으면** 프로젝트를 일시정지(Pause) 시킵니다.
- **해결**:
  1.  주기적으로(3~4일에 한 번) 앱에 접속하거나 Supabase 대시보드에 로그인하세요.
  2.  일시정지 되더라도 대시보드에서 `Restore` 버튼을 누르면 데이터 손실 없이 복구됩니다.
  3.  월 $25(Pro Plan) 결제 시 이 제한은 사라집니다.

### 🚫 Vercel 상업적 이용 제한

- **현상**: Vercel Hobby(무료) 플랜은 개인용/비영리 목적입니다. 쇼핑몰이나 유료 구독 서비스 등 **수익을 창출하는 상업 사이트**로 운영하다 적발되면 계정이 정지될 수 있습니다.
- **해결**: 수익화 모델을 도입할 때는 반드시 **Pro Plan ($20/month)**으로 업그레이드해야 합니다.

### 🔑 Google AI (Gemini) API Key

- **현상**: 현재 설정된 API Key가 무료 할당량을 초과하면 챗봇 답변이 멈출 수 있습니다.
- **확인**: [Google AI Studio](https://aistudio.google.com/)에서 사용량을 확인할 수 있습니다.

## 3. 📈 확장 가이드 (Scale-Up Guide)

사용자가 늘어나면 다음 순서로 업그레이드를 고려하세요.

1.  **1단계: Supabase Pro ($25/월)**
    - 가장 먼저 한계에 부딪히는 것은 DB 용량이나 "일시정지" 문제입니다.
    - $25를 투자하면 자동 백업, 용량 증가(8GB), 일시정지 해제 등 가장 큰 혜택을 봅니다.

2.  **2단계: Vercel Pro ($20/월)**
    - 사이트 속도가 느려지거나, 팀원과 협업이 필요할 때 업그레이드합니다.
    - 상업적 서비스를 시작한다면 필수입니다.

3.  **3단계: 요금제 최적화**
    - 이미지(Storage) 용량이 너무 크다면 AWS S3나 Cloudflare R2 같은 저렴한 스토리지로 이전을 고려합니다.
