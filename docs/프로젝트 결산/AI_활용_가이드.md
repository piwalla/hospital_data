# 🤖 AI 활용 및 질문 가이드 (AI Assistant Guide)

개발자가 아니어도 AI(ChatGPT, Claude, Cursor 등)의 도움을 받아 이 프로젝트를 수정하고 관리할 수 있습니다.
AI에게 명확하게 질문하는 방법을 알려드립니다.

## 1. AI에게 정보 제공하기 (Context)

AI는 이 프로젝트를 모릅니다. 질문하기 전에 먼저 **프로젝트의 정체**를 알려줘야 합니다.
아래 텍스트를 복사해서 질문의 첫머리에 붙여넣으세요.

```text
나는 ReworkCare라는 산재 근로자 지원 웹서비스를 운영하고 있어.
Next.js 15, Supabase, Clerk, Google Gemini AI를 사용해서 개발되었어.
나는 개발자가 아니니까 최대한 쉽고 자세하게 설명해줘.
```

**더 좋은 방법**:
Cursor나 Claude Project 기능을 사용한다면 `docs/final/` 폴더의 모든 문서(`PROJECT_OVERVIEW.md`, `TECH_STACK_ARCHITECTURE.md` 등)를 AI에게 미리 업로드해두세요. 그러면 훨씬 정확한 답변을 얻을 수 있습니다.

## 2. 상황별 질문 템플릿 (Prompts)

상황에 맞춰 빈칸만 채워서 질문하세요.

### A. 에러가 났을 때 💥

> "웹사이트에서 [어떤 행동]을 했는데 에러가 났어.
> 화면에 '오류가 발생했습니다'라고 뜨고, 브라우저 콘솔(F12)에는 아래와 같은 빨간 글씨가 보여.
>
> [에러 메시지 복사해서 붙여넣기]
>
> 문서를 참고해서 원인이 뭐고 어떻게 고쳐야 하는지 알려줘."

### B. 글자나 이미지를 바꾸고 싶을 때 ✏️

> "메인 화면에 있는 '산재 신청하기' 버튼의 글자를 '보상금 알아보기'로 바꾸고 싶어.
> 어떤 파일을 열어서 몇 번째 줄을 고쳐야 해?
> 정확한 파일 위치와 수정할 코드를 알려줘."

### C. 새로운 기능을 넣고 싶을 때 ✨

> "대시보드 화면에 '오늘의 날씨'를 보여주는 작은 상자를 추가하고 싶어.
> `dashboard` 폴더에 새 컴포넌트를 만들어서 넣으면 될 것 같은데,
>
> 1. 어떤 파일을 새로 만들어야 하는지
> 2. 그 파일에 들어갈 전체 코드
> 3. 기존 대시보드 파일(`app/dashboard/page.tsx`) 어디에 끼워 넣어야 하는지
>    단계별로 알려줘."

## 3. 추천 AI 도구 (Tools)

1.  **[Cursor](https://cursor.sh)** (강력 추천 ⭐️)
    - VS Code와 똑같이 생겼는데 AI가 내장되어 있습니다.
    - `Ctrl+L`을 눌러 채팅창을 열고 "이 파일에서 에러 좀 찾아줘"라고 하면 바로 찾아줍니다.
    - `Ctrl+K`를 눌러서 "이 함수를 좀 더 효율적으로 바꿔줘"라고 명령할 수 있습니다.

2.  **[ChatGPT](https://chatgpt.com) (Plus 모델)**
    - 파일 업로드 기능이 강력합니다. 엑셀 파일이나 PDF를 분석할 때 좋습니다.

3.  **[Claude](https://claude.ai)**
    - 코드를 굉장히 잘 짭니다. 복잡한 로직을 구현할 때 물어보면 좋습니다.
