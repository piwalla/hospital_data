# 📊 데이터 관리 가이드 (Data Management)

병원, 약국 데이터를 추가하거나 수정하는 방법을 설명합니다.

## 1. 병원 데이터 관리 (Hospital Data)

병원 데이터는 `hospitals` 테이블에 저장됩니다.

### 하나씩 추가하기 (Manual Insert)

1.  Supabase 대시보드 -> Table Editor -> `hospitals` 선택
2.  `Insert` -> `Insert row` 클릭
3.  빈칸 채우기:
    - **name**: 병원 이름
    - **address**: 주소
    - **phone**: 전화번호
    - **latitude**: 위도 (예: 37.5665)
    - **longitude**: 경도 (예: 126.9780)
    - **type**: '종합병원', '의원' 등
4.  `Save` 클릭

### 엑셀(CSV)로 한꺼번에 올리기 (Bulk Import)

1.  Supabase 대시보드 -> Table Editor -> `hospitals`
2.  `Insert` -> `Import data from CSV` 클릭
3.  준비한 CSV 파일 선택
    - **주의**: CSV 파일의 첫 줄(헤더)은 데이터베이스 컬럼 이름(`name`, `address`...)과 똑같아야 합니다.
    - CSV 파일은 반드시 **UTF-8**로 인코딩되어 있어야 한글이 안 깨집니다.

---

## 2. 게시판 관리 (Community)

악성 게시글이나 댓글을 삭제해야 할 때입니다.

1.  Supabase 대시보드 -> Table Editor -> `posts` (게시글) 또는 `comments` (댓글)
2.  내용(`content`)을 보고 문제되는 행을 찾습니다.
3.  왼쪽 체크박스 선택 -> `Delete` 버튼 클릭 -> `Confirm`
4.  **복구 불가능**하니 신중하게 삭제하세요!

---

## 3. 데이터 백업 (Backup)

혹시 모를 사고에 대비해 데이터를 백업합니다.

- **자동 백업**: Supabase Pro 요금제(유료)를 쓰면 자동으로 해줍니다.
- **수동 백업 (무료)**:
  1.  Database -> Backups -> Point in Time Recovery (PITR) 확인
  2.  또는 `Table Editor`에서 우측 상단 `Export` 버튼을 눌러서 CSV로 다운로드 받아두세요. (`hospitals`, `users` 등 중요 테이블)

---

## 4. 꿀팁: 좌표(위도/경도) 알아내기 🍯

병원 주소는 아는데 위도(`latitude`), 경도(`longitude`)를 모를 때:

1.  [구글 지도](https://www.google.com/maps) 접속
2.  병원 검색
3.  병원 위치 위에서 **마우스 오른쪽 클릭**
4.  맨 위에 나오는 숫자(예: `37.1234, 127.5678`) 클릭하면 복사됩니다.
5.  앞 숫자가 위도(lat), 뒷 숫자가 경도(lng)입니다.
