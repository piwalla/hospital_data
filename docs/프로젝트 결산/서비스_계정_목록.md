## 4. 📝 환경변수 (.env) 관리

위 서비스들의 비밀키(Secret Key)는 모두 `.env` 파일과 Vercel 대시보드에 있습니다.

- **로컬 개발**: `.env.local` 파일 확인
- **운영 배포**: Vercel Project Settings -> `Environment Variables` 확인

> **⚠️ 주의**: 절대 API Key나 Secret Key를 GitHub(공개 코드)에 올리지 마세요!
