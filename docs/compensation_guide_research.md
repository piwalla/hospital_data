# 산재 보상 항목 및 신청 단계 총정리 (Research Report)

본 문서는 근로복지공단(KCOMWEL)의 공식 급여 종류를 기반으로, 산재 환자가 **어느 단계에서 어떤 보상**을 신청해야 하는지 정리한 연구 자료입니다. 향후 웹사이트의 '보상 가이드' 및 '계산기' 기능 구현의 기초 데이터로 활용됩니다.

---

## 1. 산재 처리 단계별 보상 로드맵

산재 보상은 크게 **치료 중(요양 기간)**, **치료 종결 후(장해/복귀)**, **사망 시**로 나뉩니다.

| 단계             | 주요 상태              | 신청 가능한 보상(급여)         | 핵심 내용                                    |
| :--------------- | :--------------------- | :----------------------------- | :------------------------------------------- |
| **1. 사고/발병** | 재해 발생 직후         | **요양급여 신청**              | 산재 승인을 받기 위한 첫 단계                |
| **2. 치료 중**   | 입원/통원 치료 기간    | **요양급여**, **생활안정자금** | 병원비/간병료(현물+현금) 및 저금리 융자 지원 |
|                  | 소득 활동 불가 시      | **휴업급여**                   | 평균임금의 70% (생활비 지원)                 |
|                  | 2년 이상 미치유 시     | **상병보상연금**               | 휴업급여 대신 지급 (중증 요양 상태인 경우)   |
| **3. 치료 종결** | 의학적 치유(증상 고정) | **장해급여**, **생활안정자금** | 신체에 남은 영구적 장해 보상 & 생활자금 융자 |
|                  | 간병 필요 시           | **간병급여**                   | 치료 종결 후에도 상시 간병이 필요한 경우     |
|                  | 사회 복귀 준비 (6개월) | **재활스포츠지원**             | 요양 종결 후 6개월 내 스포츠 활동비 지원     |
| **4. 직업 복귀** | 직장 복귀 준비         | **직업재활급여**               | 훈련비용, 수당, 직장복귀지원금 등            |
| **5. 사망**      | 업무상 사유로 사망     | **유족급여**                   | 유족에게 지급 (연금/일시금)                  |
|                  | 장례 절차              | **장의비**, **생활안정자금**   | 장례비(평균임금 120일분) 및 융자 지원        |

## 2. 보상 항목별 상세 가이드

### A. 요양급여 (Medical Care Benefit) - 상세 가이드

요양급여는 산재가 승인된 후 **치료 종결(치유) 시점까지** 소요되는 의학적 비용을 보상합니다.
원칙적으로 **공단이 병원에 직접 지급(현물 급여)**하지만, 부득이하게 본인이 부담한 경우 **요양비 청구(현금 급여)**를 할 수 있습니다.

#### 1. 치료비 및 약제비 (진료비)

- **지급 규정:** 산재 승인 상병의 치료 목적 범위 내에서 인정.
  - **급여 항목:** 국민건강보험 요양급여 기준에 해당하는 진료비 전액 (본인부담금 없음).
  - **비급여 항목(본인부담):** 상급병실료(일반실 없을 때 등 예외 제외), 선택진료비, 업무와 무관한 질병 치료비, 성형/미용 목적 등은 **지원되지 않음**.
- **계산 방법:** 병원에서 공단으로 직접 청구하므로 환자가 계산할 필요 없음.
  - _Tip:_ 비급여 항목이 과다 청구된 것 같다면 공단에 **'진료비 본인부담금 확인 심사'** 요청 가능.

#### 2. 이송비 (Transportation Expenses)

- **지급 규정:** 재해 현장↔병원, 병원↔병원(전원), 통원 치료, 퇴원 시 순로(정상적인 경로)에 의한 비용.
- **지급 한도/수단:**
  - **원칙:** 대중교통(버스, 전철, 기차) 실비.
  - **택시/자가용:** 대중교통 이용이 불가능하다는 **의사 소견**이 있는 경우 인정.
- **계산 방법 (예시):**
  - **대중교통:** 영수증 실비 지급 (카드 내역 등).
  - **자가용:** (`이동 거리` x `지역별 일반 택시 요금의 50%`) + `통행료`.
    - **사설 구급차:** 이용 가능하나, 반드시 **'진료 목적의 이송'**임이 입증되어야 함.
      - _필수 서류:_ **이송처치료 영수증(일명 노란 영수증)**, 이송경로내역서, 의사 소견서(구급차 필요성 입증).
      - _지급 기준:_ 「응급의료에 관한 법률」에 따른 이송처치료 기준 실비 (부당 요금은 보건복지부 신고 후 환급 가능).
    - **택시:** 영수증 실비 (심야 할증 등 불가피한 사유 입증 필요).
- **청구 방법:** 영수증 모아서 '요양비 청구서' 제출.

#### 3. 간병료 (Nursing Fees) - 요양 중

- **지급 규정:** 입원 기간 중 부상/질병 상태가 심각하여 타인의 간병이 필수적인 경우.
  - _주의:_ 가족이 간병하더라도 청구 가능 (가족간병료). -_ **간병 등급 상세 기준 (의학적 소견 필수):**
    _ **1등급 (상시 간병):** 두 손의 손가락 상실, 두 눈 실명, 심각한 뇌 손상(혼미), 35% 이상 화상 등 일상생활 *전반*을 스스로 할 수 없어 **항상** 간병이 필요한 사람.
    - **2등급 (수시 간병):** 신경·정신·흉복부 장기에 2급 해당 장해가 남아 일상생활 *대부분*에 **수시로** 간병이 필요한 사람.
    - **3등급 (부분 간병):** 수술이나 골절 등으로 일정 기간 거동이 제한되어 부분적 도움이 필요한 사람. \* _(골절/수술 예시):_ **골반골 골절**로 수술 후 침상 안정(약 1개월 인정 사례), 하지 골절 수술 직후 등.

* **지급 금액 (1일 기준):**
  - **전문 간병인:** (1등급) 67,140원 / (2등급) 55,950원 / (3등급) 44,760원
  - **가족/기타 간병인:** (1등급) 61,750원 / (2등급) 51,460원 / (3등급) 41,170원
* **계산 예시:** 2등급 판정, 가족이 30일 간병 시 = `51,460원 x 30일` = **1,543,800원**.

#### 4. 보조기구 (Assistive Devices)

- **지급 규정:** 치료 및 재활에 필요한 보조기구를 구입했을 때 내구연한 내에서 비용 지급.
- **주요 품목 & 한도 (예시):**
  - **목발:** 알루미늄 기준 실비.
  - **휠체어:** 내구연한 5년, 기준 금액(약 48만 원) 내 실비. (초과 시 본인부담)
  - **의수/의족:** 장애 등급 및 활동성에 따라 '일반형' 기준 지급.
  - **보청기:** 내구연한 5년, 편측 131만 원 한도 (양측 인정 시 262만 원).
- **청구 방법:** 보조기구 처방전(의사) + 구입 영수증 + 검수 확인서 제출.

### B. 휴업급여 (Temporary Disability Benefit)

- **정의:** 요양으로 인해 취업하지 못한 기간에 대한 임금 보전.
- **지급액:** 평균임금의 70% (1일 기준).
  - **2025년 최고 보상 기준:** 1일 **258,132원** (이 금액을 초과해도 258,132원 지급).
  - **2025년 최저 보상 기준:** 1일 **80,240원** (단, 최저임금액보다 적으면 최저임금 적용).
- **고령자 감액 지급 (만 61세 이상):**
  - 노동 능력 감소를 고려하여 만 61세부터 지급액이 단계적으로 줄어듭니다.
  - **61세:** 66/70 (~94%)
  - **62세:** 62/70 (~88%)
  - **63세:** 58/70 (~82%)
  - **64세:** 54/70 (~77%)
  - **65세 이상:** 50/70 (~71%)
  - _예외:_ 61세 이후 취업 중 재해를 입은 경우 등은 **2년간 감액 유예**.
- **신청 시기:** 요양 기간 중 1개월 단위나 수시로 청구 가능.

### C. 장해급여 (Disability Benefit)

- **정의:** 치료가 끝난 후(치유) 신체에 영구적인 장해가 남았을 때 지급.
- **지급 기준:** 장해 등급(제1급 ~ 제14급)에 따른 **'장해보상일수'** x **'평균임금'**.
- **급여 지급 방식 (등급별 차이):**
  - **제1급 ~ 제3급:** **연금**으로만 지급. (단, 최초 1~4년분의 50%는 선급금으로 미리 수령 가능)
  - **제4급 ~ 제7급:** 연금 또는 일시금 중 **선택** 가능. (연금 선택 시 최초 1~2년분의 50% 선급 가능)
  - **제8급 ~ 제14급:** **일시금**으로만 지급.

### D. 간병급여 (Nursing Benefit) - 치료 종결 후

- **정의:** **치료가 끝난 후(치유)**에도 의학적으로 상시 또는 수시로 간병이 필요한 사람에게 지급.
  - _주의:_ '요양 중 간병료'와는 다름. 장해 등급 판정 이후의 급여.
- **지급 대상:**
  - **상시 간병급여:** 장해 1급 (신경/정신/흉복부/사지마비 등)으로 항상 간병 필요.
  - **수시 간병급여:** 장해 2급 (신경/정신/흉복부 등)으로 수시로 간병 필요.
- **지급 금액 (2024년 기준, 1일당):**
  - **전문 간병인:** (상시) **44,760원** / (수시) **29,840원**
  - **가족/기타 간병인:** (상시) **41,170원** / (수시) **27,450원**
- **신청 시기:** 요양 종결 후 장해급여 청구 시 함께 또는 별도 청구.

### E. 상병보상연금 (Injury-Disease Compensation Annuity)

- **정의:** 요양 개시 후 **2년**이 지나도 치유되지(낫지) 않고, **중증 요양 상태(폐질 등급 1~3급)**에 해당하며 취업하지 못한 경우 **휴업급여 대신 지급**하는 연금.
  - _특징:_ 휴업급여(평균임금 70%)보다 보상 수준이 높거나 유지됨.
- **지급 기준 (폐질 등급별 지급액):**

| 등급      | 상태 요약 (예시)                                            | 지급액 (1년분)                  | 비고                   |
| :-------- | :---------------------------------------------------------- | :------------------------------ | :--------------------- |
| **제1급** | **상시 간병 필요** / 양안 실명 / 양팔·양다리 절단 등        | **평균임금의 329일분** (약 90%) | 휴업급여(70%)보다 많음 |
| **제2급** | **수시 간병 필요** / 양안 시력 0.02 이하 등                 | **평균임금의 291일분** (약 80%) | 휴업급여(70%)보다 많음 |
| **제3급** | **취업 불가** / 말하기·씹기 기능 상실 / 양눈 시력 0.06 이하 | **평균임금의 257일분** (약 70%) | 휴업급여(70%)와 유사   |

- **등급별 상세 기준 (중증요양상태):**
  - **제1급:** 신경/정신/흉복부 기능 장애로 **항상 간병**이 필요한 상태, 두 눈 실명, 두 팔/다리 무릎 이상 상실 등.
  - **제2급:** 신경/정신/흉복부 기능 장애로 **수시 간병**이 필요한 상태, 두 눈 시력 0.02 이하 등.
  - **제3급:** 신경/정신/흉복부 기능 장애로 **노무 종사 불가능**한 상태, 말하기/씹기 기능 완전 상실 등.

### F. 직업재활급여 (Vocational Rehabilitation Benefit)

- **정의:** 장해를 입은 근로자의 재취업을 돕기 위해 훈련 비용과 수당을 지원.
- **신청 자격:** 장해등급 제1급 ~ 제12급 판정자 중 미취업자.
  - _기한:_ 장해 판정일로부터 **3년 이내** 신청.
  - _횟수/기간:_ 1인당 **최대 2회**, 총 **12개월** 한도.
- **지원 내용:**
  1. **직업훈련비용:** 훈련기관에 지급 (본인 부담 없음).
     - _한도:_ 1인당 **최대 600만 원** (한국장애인고용공단 위탁 시 최대 800만 원).
  2. **직업훈련수당:** 훈련 기간 중 생계비 지원 (훈련생에게 지급).
     - _지급 기준:_ **총 훈련시간을 8시간으로 나눈 일수** x **1일 최저임금액**.
     - _계산식:_ `(월 총 훈련시간 ÷ 8) x 1일 최저임금액(80,240원)`
       - _사실상 시간당 최저임금(10,030원)을 지급하는 것과 동일함._
     - _조건:_ 훈련 출석률 **80% 이상** 시 지급.
     - _감액:_ 장해연금 수령액과 합쳐 평균임금의 70%를 초과하면 그만큼 감액.

- **계산 예시 (2025년 기준):**
  - **상황:** 1일 **5시간** 수업을 듣는 A씨 (한 달 훈련일수 22일 가정).
  - **총 시간:** `5시간 x 22일` = **110시간**.
  - **환산 일수:** `110시간 ÷ 8` = **13.75일**.
  - **월 수령액:** `13.75일 x 80,240원` = **약 1,103,300원**.
  - _(사용자 의견 반영: 1일 8시간 미만 수업 시 실제 수업 시간만큼만 비례하여 지급됩니다.)_

### G. 유족급여 및 장의비 (Survivor Benefit & Funeral Expenses)

### G. 유족급여 및 장의비 (Survivor Benefit & Funeral Expenses)

- **1. 유족급여 (Survivor Benefit)**
  - **정의:** 근로자가 업무상 사유로 사망 시 유족에게 지급하는 급여.
  - **지급 방식:** **연금 지급**이 원칙. (단, 유족이 원할 경우 '일시금 50% + 연금 50%' 형태로 수령 가능).
  - **수급권자 순위 (선순위 유족에게 전액 지급):**
    1.  **배우자** (사실혼 포함)
    2.  **자녀** (만 25세 미만 또는 장애인)
    3.  **부모** (만 60세 이상 또는 장애인)
    4.  **손자녀** (만 25세 미만 또는 장애인) / **조부모** / **형제자매** 순.
  - **지급액 (유족보상연금):** `(기본금액 + 가산금액)`
    - _기본:_ 급여기초연액(평균임금×365)의 47%.
    - _가산:_ 유족 1인당 5%씩 가산 (최대 20% 한도). → **최대 평균임금의 67%**까지 지급.

- **2. 장의비 (Funeral Expenses)**
  - **정의:** 장례를 치른 사람(유족 등)에게 지급하는 장례 비용 지원금.
  - **지급액:** **평균임금의 120일분**.
  - **2025년 상한/하한액 고시 (제한 사항):**
    - **최고 한도:** **18,685,600원** (이 금액 이상은 못 받음).
    - **최저 보장:** **13,451,380원** (평균임금이 낮아도 이만큼은 보장).
    - _(참고: 매년 고용노동부 고시로 금액 변동)_

### H. 재활스포츠지원 (Rehabilitation Sports Support)

- **정의:** 산재 근로자의 신체 기능 회복과 사회 복귀를 돕기 위한 스포츠 활동 비용 지원.
- **지원 대상:** 요양 중이거나 요양 종결 후 **6개월 이내**인 산재근로자.
  - _조건:_ 장해 판정(예상)자로서 주치의 소견이나 공단 자문의 인정 필요.
- **지원 내용:**
  - **일반 재활스포츠:** 월 **10만 원** 이내 (최대 3개월).
  - **특수 재활스포츠:** 월 **60만 원** 이내 (최대 1개월).

### I. 산재근로자 생활안정자금 융자 (Loans)

- **정의:** 산재로 인해 생활이 어려운 근로자에게 저금리로 생활자금을 빌려주는 제도.
- **신청 자격 (상세):**
  - **소득 기준:** 월 평균 소득이 **3인 가구 중위소득 이하** (2025년 기준 월 약 502만 원 이하).
  - **대상 상태:**
    1.  **요양 중:** 3개월 이상 요양 중인 산재근로자 (월 소득이 최저임금 이하인 경우 등).
    2.  **요양 종결:** 장해등급 **제1급 ~ 제9급** 판정자 (상병보상연금 수급자 포함).
    3.  **유족:** 산재 사망 근로자의 유족 (수급권자).
  - **제외 대상:** 신용불량자(연체 등), 국세 체납자 등은 제한될 수 있음.
- **신청 기한 (사유 발생일 기준):**
  - 각 융자 종류별로 신청 기한이 다름 (예: 혼례/장례/차량구입일로부터 **90일 이내** 등).
  - _주의:_ 기한을 넘기면 신청 불가하므로 사유 발생 즉시 확인 필요.
- **융자 조건:**
  - **금리:** 연 **1.0%** (2025년 한시적 인하, 별도 보증료 0.9%~1.0% 있음).
  - **상환:** 2년 거치 3년 분할 상환 등.
- **주요 융자 종류 및 한도 (1세대당 최대 3,000만 원 한도):**
  - **의료비/혼례비/장례비:** 각 1,000만 원 한도.
  - **차량구입비:** 1,500만 원 한도 (본인 명의, 사업용 제외).
  - **주택이전비:** 1,500만 원 한도 (전용면적 85㎡ 이하).
- **신청 방법:**
  - **온라인:** [근로복지넷](https://welfare.comwel.or.kr) (공동인증서 필요).
  - **방문:** 근로복지공단 지사 방문 접수.
  - **실행:** 융자 결정 후 **우리은행** 인터넷뱅킹/앱을 통해 대출 실행.

## 3. 웹사이트 '보상 항목' 구현 계획 (제안)

MVP 단계에서 구축된 기능(평균임금, 휴업, 장해) 외에 추가해야 할 콘텐츠 구조 제안입니다.

### [Phase 1] 보상 가이드 페이지 신설

- **메뉴 구조:** `정보 가이드` > `산재 보상 종류`
- **구성:** 위의 "단계별 로드맵"을 시각화(Timeline UI)하여 보여줌.
- **인터랙션:** 각 단계 클릭 시 상세 설명 모달 또는 아코디언 열림.

### [Phase 2] 누락된 계산기/가이드 추가 연결

1.  **휴업급여 계산기:** (기존 완료) -> 결과 페이지에서 `요양급여(이송비)` 안내 배너 추가.
2.  **장해급여 계산기:** (기존 완료) -> 결과 페이지에서 `간병급여` 및 `직업재활` 안내 추가.
3.  **신규 콘텐츠:**
    - **[서류 가이드] 이송비 청구 방법:** 이송비 영수증 모으는 팁.
    - **[서류 가이드] 간병료 청구 방법:** 간병인 자격 요건 등.

### [Phase 3] 랜딩 페이지 연결 (Feature A)

- **링크:** "나에게 맞는 보상 항목" 카드 클릭 시 -> `Phase 1`의 **[나만의 보상 로드맵]** 페이지로 이동.
- **기능:** 사용자 상태(치료중/종결 등)를 선택하면 받을 수 있는 급여만 필터링해서 보여주기.
