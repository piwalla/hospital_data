## 3. 구현 전략 (A안)

### 1단계: 승인율 (Approval Rate) 계산

사용자에게 가장 중요한 "내 케이스의 승인 확률"을 제공하기 위해 **상병별 신청/승인 데이터**를 비교합니다.

1. `OPA020MT_11_INFO` (신청 상병별) 호출 → 해당 상병의 전체 신청 건수 ($A$) 확보
2. `OPA020MT_21_INFO` (승인 상병별) 호출 → 해당 상병의 승인 건수 ($B$) 확보
3. **승인율** = $(B / A) * 100$

### 2단계: 예상 소요 기간 (Duration)

`OPA020MT_04_INFO` (승인 통합 정보)를 샘플링하여, `재해발생일`과 `요양승인(결정)일` 간의 평균 차이를 계산합니다.

## 4. 테스트 계획 (Curl)

가장 핵심적인 **승인 통합 정보**(`OPA020MT_04_INFO`)를 먼저 호출하여 데이터 구조를 확인합니다.

curl "http://apis.data.go.kr/B490001/SJBoheomYoyang/OPA020MT_04_INFO?serviceKey=[KEY]&pageNo=1&numOfRows=10&type=json"
